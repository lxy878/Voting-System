# Voting System Developing Environment
* OS: MacOS
* IDE: IntelliJ

## Backend
* Language: Java 11
* Framework: Spring Boot
* In-memory Database: H2
* Project: Maven

### Run with Maven
```
mvn build
```
```
mvn run
```

### Test with Maven
```
mvn Test
```

## Frontend
* Language: Javascript
* Framework: Reactjs
* In-memory: Redux
* Other: Nodejs

## Run with npm
```
 npm install 
```
```
 npm start
``````
