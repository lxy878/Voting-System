# Voting System Developing Environment
* OS: MacOS
* IDE: IntelliJ
## Backend
* Language: Java 11
* Framework: Spring Boot
* In-memory Database: H2
* Project: Maven
## Frontend
* Language: Javascript
* Framework: Reactjs
* In-memory: Redux
* other: Nodejs

## Running Frontend
### with npm
```
 npm install 
```
```
 npm start
```

## Running with Backend
### with Maven
```
mvn build
```
```
mvn run
```
